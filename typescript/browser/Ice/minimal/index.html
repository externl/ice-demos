Minimal Demo Readme

This demo is a minimal Ice "hello world" application. Each time the client is run a "sayHello" invocation is sent to the server.

To run the demo, first you need to start an Ice minimal server from another language mapping (Java, C++, C#, or Python) with web socket support. For example with Python, run: 

    python Server.py --Ice.Default.Protocol=ws
Please refer to the README.md in the server subdirectory for more information on starting the server.

After starting the server, run the client by clicking the "Hello World!" button.

Note that this demo hardwires port 10,000. If port 10,000 is not available on your machine, you need to edit both client and server to use a free port.
